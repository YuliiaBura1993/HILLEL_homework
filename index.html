Селекторы CSS

In CSS, selectors are patterns used to select the element(s) you want to style.

№ Selector Example Example description
1 .class .intro Selects all elements with class="intro"
2 .class1.class2 .name1.name2 Selects all elements with both name1 and name2 set within its class attribute
3 .class1 .class2 .name1 .name2 Selects all elements with name2 that is a descendant of an element with name1
5 * * Selects all elements
6 element p Selects all p elements
7 element.class p.intro Selects all p elements with class="intro"
8 element,element div, p Selects all div elements and all p elements
9 element>element div > p Selects all p elements where the parent is a div element
10 [attribute] [target] Selects all elements with a target attribute

Типы селекторов

Селектор определяет, к какому элементу применять то или иное CSS-правило.

  • По названию. Селектор типа иногда называется селектором имени тега или селектором элемента, поскольку он выбирает тег/элемент HTML в вашем документе;
  • По id. Селектор ID начинается с #, а не с точки, но используется так же, как и селектор класса. Однако ID может быть использован единожды на странице, и к элементу может быть применён только один id;
  • По классу. Селектор класса начинается с символа точки (.). Он выберет в документе всё, к чему применён этот класс;
  • По нескольким классам. Вы можете применить несколько классов к элементу и нацелиться на них по отдельности, или выбрать элемент только если присутствуют все классы селектора;
  • По вложенности. Можно комбинировать любые типы селекторов через пробел. Такие селекторы называются вложенными или контекстными и читаются справа налево.

Селекторы атрибутов

Селекторы атрибутов отбирают элементы по наличию атрибута или его значению.

Polski: Dzień dobry!
English: Good afternoon!
